# qBrowser
<img width="1421" height="886" alt="png" src="https://github.com/user-attachments/assets/27dc1a1a-6f4c-4083-9955-714b48390e7d" />
A basic browser written in Python, without cookies. 
A couple of advantages: nothing is saved or tracked except for the websites you visit, the lack of cookies provides some anonymity after you close the browser, and the interface is fully customizable.
